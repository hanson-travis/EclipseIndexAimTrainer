# Eclipse Index Trainer

A precision billiards training tool for mastering the **Eclipse Index (EI)** aiming system.

## Features
- **Plan View**: Top-down diagram showing the cue ball, object ball, ghost ball, and pocket target.
- **Shooter View**: Point-of-view perspective from behind the cue ball.
- **Interactive Training**: Drag the object ball in the shooter view to match the required cut angle.
- **Instant Feedback**: Learn to recognize specific EI values (0, 2, 4, 6, 7, 8) and their corresponding cut angles.

## The EI System (Level 1)
| EI | Thickness | Cut Angle | Description |
|---|---|---|---|
| 0 | 1/1 (Full) | 0° | Straight shot |
| 2 | 3/4 | ~14.5° | Slight cut |
| 4 | 1/2 | 30° | Standard cut |
| 6 | 1/4 | ~48.6° | Thin cut |
| 7 | 1/8 | ~61.0° | Very thin |
| 8 | Edge | 90° | Near-miss (Extreme) |

## Tech Stack
- React 18
- TailwindCSS
- HTML5 Canvas for rendering

## Usage
Simply open `index.html` in a web browser. No build process required.
